## Project Overview
This project performs Visual Exploratory Data Analysis (EDA) on two datasets:
- Airbnb Listings Dataset
- FIFA World Cup Matches Dataset (1930–2014)

The goal is to analyze patterns, trends, and relationships using data visualization techniques.

---

## Objectives
- Perform univariate, bivariate, and multivariate analysis  
- Visualize distributions and relationships  
- Validate hypotheses using graphical methods  
- Construct a network graph for FIFA World Cup 2014 matches  

---

## Airbnb Dataset Analysis
### Hypothesis:
Entire homes/apartments are more expensive than private/shared rooms, and property type also influences pricing.

### Techniques Used:
- Boxplots
- Histograms
- Countplots
- Scatter plots
- Heatmaps
- Bar plots

### Key Findings:
- Entire homes have the highest prices  
- Price distribution is right-skewed  
- Property type significantly impacts pricing  

---

## World Cup Dataset Analysis
### Hypothesis:
Host nations had stronger performance in early decades compared to modern tournaments.

### Techniques Used:
- Line plots
- Bar plots
- Heatmaps
- Stacked bar charts

### Key Findings:
- Host advantage was stronger in early decades  
- Performance of host nations declined over time  
- Modern tournaments show less home advantage  

---

## Network Graph (2014 World Cup)
- Nodes represent teams  
- Edges represent matches  
- Graph shows relationships and match interactions between teams
